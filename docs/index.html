Description:

The Asterism Custom Select JavaScript library allows to create custom select boxes with item filtering support. By default, the Asterism Custom Select automatically add a filter search input to the select box when the number of options are larger than 7.

How to use it:

Include the select.min.css and select.min.js on the webpage.

view source
1 <link href="dist/select.min.css" rel="stylesheet">
2 <script src="dist/select.min.js"></script>

Let’s say you have a native select box like this:

view source
01 <select id="demo">
02   <option value="1">HTML5</option>
03   <option value="2">CSS3</option>
04   <option value="3">JavaScript</option>
05   <option value="4">jQuery</option>
06   <option value="5">AngularJS</option>
07   <option value="6">ReactJS</option>
08   <option value="7">React Native</option>
09   <option value="8">Bootstrap</option>
10   <option value="9">Vue.js</option>
11   <option value="10">Python</option>
12 </select>

Create a new Select object and specify the target element.

view source
1 var mySelect = new Select('#demo');

To config the Asterism Custom Select, just pass the following options to the Select object.

view source
01 var mySelect = new Select('#demo',{
02  
03     // auto show the live filter
04     filtered: 'auto',
05  
06     // auto show the live filter when the options >= 8
07     filter_threshold: 8,
08  
09     // custom placeholder
10     filter_placeholder: 'Filter options...'
11      
12 });
